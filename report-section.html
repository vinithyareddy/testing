What is the Prompt-a-thon?

The Prompt-a-thon is an interactive learning experience designed to help you master querying Scorecard data using natural language. You'll work with AI-powered tools to transform how you access, understand, and leverage Scorecard insights for your role.

This hands-on workshop combines guided tutorials, pre-built prompt templates, and real-time feedback so you can ask better questions, interpret results confidently, and trace exactly how each answer was produced.

What you'll accomplish

  • ✓ Master natural language queries: Ask questions in plain English and get instant answers grounded in Scorecard data
  • ✓ Use the prompt library: Customize role-based templates or create your own prompts from scratch
  • ✓ Understand data provenance: See the SQL, filters, and tables behind each result ("How computed")
  • ✓ Apply filters effectively: Fiscal year, organization, region, sector, disaggregation, and more
  • ✓ Navigate ADM compliance: Learn governance constraints and compliant ways to ask questions
  • ✓ Provide feedback: Rate responses and help improve the system

How it works

  1. Tell us about your role and choose your segment to personalize prompts and guidance
  2. Open the Prompt Library (left sidebar) and drag-and-drop prompts into the chat
  3. Ask questions and iterate with follow-up prompts ("compare", "drill down", "explain why")
  4. Review "How computed" to validate filters, SQL, and data sources
  5. Save/share outputs and tag feedback to improve the experience

Important: Data scope & filters

This prototype may be limited to a vetted FY25 snapshot. Also, filters can auto-apply (e.g., region/sector/disaggregation). If you get empty results, clear filters or broaden them before re-asking.
